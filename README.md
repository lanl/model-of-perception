# NeRF Visualization and Analysis

This repository contains code for Neural Radiance Fields (NeRF) visualization and analysis, including Cinema database generation, reconstruction scripts, and various plotting utilities.

## Components

- **NeRF IMDB**: Tools for creating Cinema databases from NeRF data
- **Reconstruction**: Scripts for 3D reconstruction from NeRF models
- **Visualization**: Various plotting and visualization utilities
- **Cinema Integration**: PyCinema and Synema integration for interactive visualization

## Key Scripts

- `reconstruct.py` - Main reconstruction script
- `plotL2.py` - L2 error plotting and analysis
- `align.py` - Data alignment utilities
- `viewScalar.py`, `viewer.py` - Visualization tools

## Dependencies

- NumPy
- Matplotlib
- SciPy
- PyVista
- VTK
- PyTorch
